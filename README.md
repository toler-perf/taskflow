# 📝 TaskFlow

> Лёгкий менеджер задач с категориями, мгновенным обновлением и базовой статистикой.

## 🚀 Быстрый старт

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/toler-perf/taskflow.git
   cd taskflow
   ```

---

### 🐳 2. Финальный запуск Docker

Выполните в корне проекта:


```bash
# Полная пересборка и запуск всех 3 сервисов
docker compose down
docker compose up -d --build
```
